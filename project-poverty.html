Predicting Multidimensional Poverty in Argentina

Development of a predictive framework using Machine Learning techniques to estimate multidimensional poverty from household survey data.

Overview

This project was developed as my Master's Thesis in Computational Social Science. The objective was to build an interpretable machine learning framework capable of predicting multidimensional poverty using data from Argentina's Permanent Household Survey (EPH).

Objectives

  • Predict multidimensional poverty using household survey data.
  • Compare several machine learning algorithms.
  • Identify the most influential predictors.
  • Create a reproducible analytical workflow.

Methodology

The workflow included data cleaning, feature engineering, exploratory data analysis, model training, hyperparameter tuning and model evaluation.
